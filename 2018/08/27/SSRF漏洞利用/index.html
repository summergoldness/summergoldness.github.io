weblogic SSRF漏洞利用

WebLogic SSRF漏洞简介

漏洞编号：CVE-2014-4210
漏洞影响：版本10.0.2,10.3.6
漏洞产生的原因：
Weblogic中存在一个SSRF漏洞，利用该漏洞可以发送任意HTTP请求，进而攻击内网中redis、fastcgi等脆弱组件。
复现借鉴文档：Weblogic SSRF漏洞 @Vulhub
Oracle WebLogic Web Server既可以被外部主机访问，同时也允许访问内部主机。可以用来进行端口扫描等。
